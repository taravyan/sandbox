The < b > tag. < strong > should be used instead, or font-weight: bold in CSS.
The < i > tag. < em > should be used, or font-weight: italic in CSS.
The < big > tag. < h1 > etc. should be used, or font-size in CSS.
The < small > tag. font-size should be used in CSS.
The < hr > tag. Using a border-top or border-bottom in CSS would be better.
The < u > tag, as nothing but links should be underlined.
The < center > tag. text-align should be used in CSS.
The < menu > tag. Use < ul > instead.
The < layer > tag.
The < blink > or < marquee > tags.
The < font > tag. CSS instead.
The name attribute. MOSTLY just use id instead.
The text and bgcolor attributes in the body. Use the color and background-color properties in CSS.
The background attribute in the body. Use background-image in CSS.
The link, alink, and vlink attributes in the body. Use :link, :active, and :visited in CSS.
The align attribute. Use CSS.
The target attribute. Users will decide when they want to open new windows.
Tables.
Frames.
